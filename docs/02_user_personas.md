## 페르소나 1 (Primary): 해외 취업 준비 간호사

**이름**: 김소연, 28세  
**직업**: 국내 대학병원 ICU 간호사, 경력 4년  
**상황**: NCLEX-RN 합격 후 시드니 병원 취업 확정. 4개월 후 출국 예정.

### 배경
- 근무 형태: 3교대 (주간/오후/야간)
- 학습 가능 시간: 출퇴근 지하철 10-15분, 식사 후 휴식 5-10분
- 영어 수준: 독해 중급, 구어 및 임상 소통 취약
- 스마트폰: iPhone 14

### 페인포인트
1. 교과서 영어와 실제 워드 회화 사이의 간극 — "NPO"는 알지만 환자에게 설명하는 방법을 모른다
2. 환자 갈등 상황 대처 불안 — 퇴원 요구, 치료 거부, 불만 표현 등을 어떻게 다뤄야 할지 두렵다
3. 학습 지속의 어려움 — 교대 근무로 규칙적인 스케줄이 없어 앱을 꾸준히 열기 힘들다
4. 단조로운 단어 암기에 빠른 흥미 상실

### 목표
- 호주 병원 첫 출근일에 자신 있게 환자, 동료와 소통하고 싶다
- 특히 비협조적 환자 상황에서 전문적이고 침착하게 대응하고 싶다
- 출국 전까지 "할 수 있다"는 자신감을 갖고 싶다

### 사용 패턴
- 세션 길이: 10-15분
- 접속 빈도: 주 4-5회
- 동기 유발 요소: 진도 가시화, 스트릭 유지, 수집/꾸미기 (가챠 메커니즘에 친숙)
- 이탈 트리거: 같은 형식의 문제 반복, 학습 방향이 보이지 않을 때

### forin에서의 기대
- "오늘 이 상황에서 이런 말을 배웠다"는 즉각적 현장 연결감
- 귀여운 고양이 캐릭터를 꾸미는 재미
- 14일 스트릭 유지 성취감

## 페르소나 2 (Secondary): 영국 이직 준비 의사

**이름**: Arjun Sharma, 34세  
**직업**: 인도 GP (일반의), 경력 7년  
**상황**: 영국 PLAB 시험 준비 중. 합격 후 Birmingham 병원 임상 포스팅 예정.

### 배경
- 근무 형태: 주중 클리닉 (바쁜 일정)
- 학습 가능 시간: 주말 30-45분 집중 세션
- 영어 수준: 의학 문서/논문 영어 강함, 환자 대면 구어체 취약
- 스마트폰: Samsung Galaxy S24

### 페인포인트
1. 방퇴원서, 레퍼럴 레터는 잘 쓰지만 환자와의 일상 임상 대화에서 막힌다
2. 나쁜 소식 전달(breaking bad news), 동의 획득(informed consent) 대화에 적합한 어조를 모른다
3. NHS 특유의 워크플로우 용어, 영국식 임상 표현에 익숙하지 않다
4. 게임화보다는 콘텐츠 정확성과 깊이를 우선시한다 — 임상적으로 잘못된 언어 예시를 보면 신뢰를 잃는다

### 목표
- PLAB 언어 영역 대비
- 영국 환자와의 민감한 대화(나쁜 소식, 동의, 불만)를 품위 있게 처리하고 싶다
- NHS 워크플로우를 사전에 이해하고 싶다

### 사용 패턴
- 세션 길이: 주말 30-45분
- 게임화보다 콘텐츠 깊이 선호
- 이탈 트리거: 임상 표현의 정확도 의심, 너무 단순한 문제

## 페르소나 3 (Tertiary): 캐나다 취업 약사

**이름**: 박민지, 26세  
**직업**: 국내 대학 약학부 졸업, 신입  
**상황**: 캐나다 온타리오주 병원 약국 취업 확정.

### 배경
- 영어 경험: 학교 시험 영어만, 실제 대화 경험 전무
- 학습 가능 시간: 매일 20-30분 (취업 준비 기간이라 시간 여유 있음)
- 스마트폰: Android

### 페인포인트
1. 약 이름 발음에 대한 공포 ("hydrochlorothiazide를 어떻게 말하지?")
2. 환자 상담(카운터 서비스) 경험이 한국에서도 없었는데 영어로 해야 한다는 부담
3. 캐나다 환자와의 커뮤니케이션 문화 차이를 모른다
4. 초급 영어 학습자라 의료 용어 위주 콘텐츠는 너무 어렵게 느껴진다

### 목표
- 기본 환자 상담 어휘 습득
- 약물 상호작용 설명 언어 연습
- 매일 꾸준히 학습해서 출국 전에 자신감 갖기

### 사용 패턴
- 세션 길이: 20-30분
- 접속 빈도: 거의 매일
- 동기 유발 요소: 게임화 + 보상에 강하게 반응
- 필요 조건: 충분한 스캐폴딩, 천천히 오르는 난이도 커브

## 사용자 여정 맵 (소연 — Primary Persona)

```
인식 단계
 └─ NCLEX 합격 후 "이제 영어 말하기 실력이 문제다" 인식
    → 앱스토어 검색 "간호사 영어" → forin 발견

검토 단계
 └─ 스크린샷에서 "실제 병원 대화 시나리오" 확인
    → "듀오링고 같은데 간호사 전용?"이라는 기대감으로 설치

온보딩 단계
 └─ 직업 선택: 간호사 → 국가: 호주 선택
    → 레벨 평가: 6/10 맞혀서 Intermediate 배정
    → 고양이 이름 "나비" 설정 → 첫 아이템 획득 → 강한 첫인상

학습 단계
 └─ 매일 출퇴근 지하철 2스테이지씩
    → 14일 스트릭 달성 → Silver 상자 획득 → 고양이 간호사복 장착
    → "이 상황 실제 현장에서도 쓸 수 있겠다" 체감

전도 단계
 └─ 동료 간호사에게 "이거 진짜 현장 영어야" 추천
    → 카카오톡 링크 공유
```
