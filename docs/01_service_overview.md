## 미션

> "forin은 해외 취업을 준비하는 의료 전문가들이, 실제 임상 현장에서 자신 있게 소통할 수 있도록 현장 시나리오 기반의 게임화된 모바일 언어 학습을 제공한다."

---

## 문제 정의

듀오링고, 바벨 등 범용 언어 학습 앱은 해외 취업 의료인이 직면하는 실제 문제를 해결하지 못한다.

| 범용 앱이 가르치는 것 | forin이 가르치는 것 |
|---|---|
| "화장실이 어디예요?" | "아직 트로포닌 수치가 안 나왔는데 퇴원하시면 위험할 수 있습니다." |
| "이름이 뭐예요?" | "환자분의 약물 알레르기 이력을 확인해도 될까요?" |
| "날씨가 좋네요." | "지금 바로 담당 의사에게 escalation이 필요합니다." |

해외 취업 간호사가 호주 병원에 첫 출근하는 날, 필요한 것은 일상 대화가 아니다.

- 퇴원 거부 환자 응대
- 약물 조정 대화
- SBAR/ISBAR 프로토콜 기반 인수인계
- 긴급 상황 팀 커뮤니케이션
- 문화적으로 적절한 informed consent 설명

forin은 이 간극을 메운다.

## 서비스 개요

**앱 이름**: forin  
**타겟 플랫폼**: iOS, Android  
**1차 타겟 사용자**: 해외 취업 준비 간호사 (이후 의사, 약사로 확장)  
**학습 언어**: 영어 (MVP) → 일본어, 독일어 (Phase 3)  
**목표 국가**: 호주, 영국, 미국, 캐나다, 뉴질랜드

### 핵심 학습 방식
듀오링고의 게임화 구조를 직업 특화 임상 시나리오에 접목한다.
- 현장 기반 시나리오 (실제 병원에서 발생하는 상황)
- 4가지 인터랙티브 문제 유형 (문장 배열 / 빈칸 채우기 / 의미 매칭 / 대화 연습)
- 단계별 잠금 해제 (커리큘럼 진행감)
- 고양이 캐릭터 꾸미기 (성취 보상, 재방문 동기)

---

## 비전

**3년 목표**: 해외 취업 의료 전문가를 위한 언어 학습 플랫폼의 글로벌 스탠다드가 된다.
- 5개 이상 언어 지원
- 10개 이상 대상 국가
- 10개 이상 의료 직군 커버

---

## 핵심 가치

| 가치 | 설명 |
|------|------|
| **현장성** | 교과서 영어가 아닌, 실제 워드에서 들리는 영어를 가르친다 |
| **안전성** | 언어 장벽으로 인한 의료 사고를 예방한다 |
| **성취감** | 작은 진전도 시각적으로 체감할 수 있게 한다 |
| **효율성** | 교대 근무자도 10분 세션으로 꾸준히 학습할 수 있다 |

## 비즈니스 모델

| 구분 | 내용 |
|------|------|
| **Freemium** | 기본 학습 무료, 목숨 제한 / 광고 포함 |
| **Premium 구독** | ₩9,900/월 또는 $7.99/월 — 무제한 목숨, 광고 없음, 전체 커리큘럼 접근 |
