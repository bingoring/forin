## 2. 일일 학습 플로우

```
홈 화면
  │
  ├── 일일 진도 바 표시 (예: 0/2 스테이지)
  ├── 현재 모듈 + 다음 스테이지 카드 하이라이트
  ├── 목숨(하트) 표시
  └── 스트릭 카운터
  │
  ▼ [계속하기] 또는 스테이지 카드 탭
  │
┌─────────────────────────────────┐
│      스테이지 소개 화면          │
│  제목: "환자 퇴원 요구 대처"     │
│  예상 소요: 약 5분               │
│  ★ 기회: 3별 가능               │
│  [시작]                         │
└──────────────┬──────────────────┘
               │ POST /learning/stages/:id/start
               ▼
┌─────────────────────────────────┐
│     문제 진행 (1/4)              │
│                                 │
│  상단: 시나리오 요약             │

│                                 │
│  [문장 배열 UI]                  │

│  [확인]                         │
└──────────────┬──────────────────┘
               │ 제출
               │
        ┌──────┴───────────┐
        ▼                  ▼
     정 답               오 답
  ──────────          ──────────

               │ POST /attempts/:id/complete
               ▼
    ┌──────────────────────┐
    │      레벨업?          │
    │  YES → 레벨업         │
```

## 7. 화면 네비게이션 구조

```
Bottom Tab Navigation
  │
  ├── 🏠 홈 (Home)
  │     ├── 현재 모듈/스테이지 카드
  │     ├── 일일 진도 바
  │     └── 스트릭 카운터
  │
  ├── 📚 학습 (Learn)
  │     ├── 전체 커리큘럼 맵
  │     ├── 모듈 목록
  │     └── 유닛 → 스테이지 드릴다운
  │
  ├── 🏆 업적 (Achievements)
  │     ├── 달성 업적 목록
  │     └── 미달성 업적 (잠금 아이콘)
  │
  └── 👤 프로필 (Profile)
        ├── 고양이 커스터마이징
        ├── 인벤토리
        ├── Cat Shop
        ├── 학습 통계 (주간/월간)
        └── 설정 (알림, 목표, 계정)
```
